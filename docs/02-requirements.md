# 02 - Requirements

## 功能需求（Functional Requirements）

### 1.1 用户系统
- 用户注册 / 登录（邮箱/手机）
- 用户资料：头像、昵称、简介、兴趣
- 关注 / 取关 / 粉丝列表
- 权限系统（用户、管理员）

### 1.2 内容系统（帖子）
- 发布文本内容
- 发布图文帖子
- 发布视频内容
- 发布攻略（长文）
- 点赞 / 收藏 / 评论
- Feed 流（按时间 / 热度排序）

### 1.3 媒体处理
- 图片压缩、缩略图生成
- 视频转码 / 截图 / 封面提取
- 音频波形提取
- 异步处理任务（通过 Kafka）

### 1.4 AI 模块
- 音频分离（人声/伴奏）
- AI 人声音色模仿
- AI 修图 / 增强
- 自动生成短视频（图片 + 音频）

---

## 2. 非功能需求（Non-functional Requirements）
- 高并发可扩展（媒体上传）
- 高性能（缓存 + MQ）
- 可用性（网关 + 熔断）
- 可维护性（模块划分清晰）
- 可观测性（日志、链路追踪）
